# DisableHelmetArmor
Disable All Helmet Armor in FiveM

# How to install
- Download the DisableHelmetArmor folder and place it in your FiveM resources folder.
- Ensure DisableHelmetArmor in your server.cfg file.
- Restart your FiveM server.
